# Orbit v0.1 Open Source Release Design

## Goal

Prepare Orbit v0.1 for a credible open-source launch under
`github.com/TheDartsCo/orbit`.

Orbit v0.1 is a macOS-first release. The app may build elsewhere, but the
project will not claim Linux or Windows support until those platforms are
tested.

## Release Scope

The release package includes:

- A new Orbit identity and complete Tauri app icon set
- A direct, product-focused `README.md`
- A practical `CONTRIBUTING.md`
- An MIT `LICENSE` owned by The Darts Co
- Correct package, Cargo, and Tauri metadata
- A real application screenshot for the README

Automated releases, signing, notarization, and broader platform support are
outside this pass.

## Visual Identity

The icon represents coding-agent sessions converging into one searchable
center.

- Rounded-square macOS app icon
- Deep near-black background
- Central index/search node connected to surrounding session nodes
- Connections suggest an orbital `O` without using a literal atom symbol
- Electric blue, violet, magenta, and coral gradient
- Simple geometry with restrained glow and depth

The identity should feel like a serious developer tool using current AI-era
colors, not an "AI magic" product.

Deliverables:

- Master SVG
- Transparent GitHub/README mark
- Tauri-generated platform icon set

## Documentation

The README leads with the user problem:

> Your coding-agent history is scattered across tools. Orbit puts it in one
> place.

It will include:

- Compact branded header and real screenshot
- Clear macOS-first status and local-data privacy statement
- Concrete capabilities
- Supported adapter table with honest notes
- Installation from GitHub Releases and from source
- Short usage flow
- Architecture summary for contributors
- Known v0.1 limitations
- Links to contribution guidance and the license

The writing will be direct and specific. It will avoid a badge wall, generic
community language, an aspirational roadmap, and unsupported claims.

`CONTRIBUTING.md` will cover local setup, verification commands, adapter
expectations, and useful pull-request scope.

## Metadata

Release metadata will consistently use:

- Product name: Orbit
- Version: 0.1.0
- Repository: `https://github.com/TheDartsCo/orbit`
- Copyright holder: The Darts Co
- License: MIT
- Positioning: macOS-first local coding-agent session browser
